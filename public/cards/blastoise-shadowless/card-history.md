# Shadowless 거북왕 카드 역사

## Shadowless란?

1999년 Base Set 초기 인쇄본:
- 카드 오른쪽에 그림자 효과 없음
- 1st Edition과 Unlimited 사이 과도기
- 1st Edition Shadowless가 가장 희귀

## 리자몽 대비 위치

거북왕은 항상 리자몽의 그림자:
- 같은 희소성, 절반 가격
